# The Alter Office - Round 2 Assessment

Mini project assessment for The Alter Office Round 2 recruitment.

## Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: MongoDB (Mongoose)

## Setup

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```

## Environment Variables

Create a `.env` file in the `server` folder:
```
MONGO_URI=your_mongodb_connection_string
PORT=5000
```
